Q1 Difference between “ == “ and “ === “ operators.

The main difference between the == and === operator in javascript is that the == operator does the type conversion of the operands before comparison, whereas the === operator strictly compares the values as well as the data types of the operands.
If the two values are not equal and of the same type , it will return false.

Q2 What is the spread operator?

The spread operator allows us to copy some part or all parts of an existing array or object into another array or object

Q4 What is execution context?

  • Execution context is the one in which the current code actually runs and gets executed.
  • It decides which code section has the access to the functions, variables, and ojects used in the code.
  • If the current execution context doesn't contain the declared variable, then it looks for it in the parent execution context.
  • After the code gets executed within the execution context, the context gets erased.

Q5 What is meant by first class functions

First-class functions are functions that are treated as variables. They can also be parsed into other functions as arguments.

The functions can be assigned to any other variable or passed as an argument or can be returned by another function.

Q6 What are closures? Give an example of closure

A closure is the combination of a function bundled together (enclosed) with references to its surrounding state (the lexical environment). In other words, a closure gives you access to an outer function scope from an inner function. In JavaScript, closures are created every time a function is created, at function creation time.

Explain call(), apply() and, bind() methods. Give an example of call(), apply(), bind()

  • Call():calls a function with a given this value and arguments provided individually.
  • Apply():call() and apply() serve the exact same purpose. The only difference between how they work is that call() expects all parameters to be passed in individually, whereas apply() expects an array of all of our parameters.
  • Bind():The bind() method creates a new function that, when called, has its this keyword set to the provided value. (Using Bind an object can borrow a method from another object.)
