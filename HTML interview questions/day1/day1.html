Q1 Difference between div and span

Ans: A div is a block level element whereas a span is an inline element.
The div is used to wrap sections of a document, while we use spans to wrap a portion of text or image

Q2 Difference between class and IDs

Ans: Each element can have only one ID whereas you can use the same class on multiple elements. ID name starts with the “#” symbol followed by a unique name assigned to it. On the other hand class assigned to an element has its name starts with “.” followed by class name.
The class can be applied to multiple elements so it could be multiple times on a single page.

Q3 What are self closing tags

Ans: Self closing tags in HTML are tags that need not be closed. They are also called empty tags.
ex: "img" or "br" or "link" tags

Q5 Difference between block and inline

Ans: Block level elements are those which occupy 100% width of the webpage or browser whereas the inline element are those which occupy only the amount of width they take.
Block level Elements always starts from the next line whereas the inline elements always starts from the same line.

Q8 Difference between HTML tags and elements

Ans: Tags are the starting and ending parts of an HTML element. They begin with < symbol and end with > symbol. Whatever written inside < and > are called tags.

Elements enclose the contents in between the tags. They consist of some kind of structure or expression. It generally consists of a start tag, content and an end tag.

Q9 Explain br tags

Ans: A br tag is a self closing or an empty tag that is used to create a horizontal break between two html elements or within two html elements

Q10 What are semantic tags?

Ans Semantic tags are those tas in HTML that are used to define different parts of a web page. It makes the content more comprehensible by better defining the different sections and layout of web pages.
ex: "header", "footer", "main"
