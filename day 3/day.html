Q1 Difference between flex row and column

Flex row is a property of flex which is used to display the elements in row whereas flex column is used to display the elements in cloumn.

Q2 Explain inline , internal and external stylesheet

You can apply CSS to your webpage's HTML files using an inline style, internal stylesheet, or external stylesheet.
Inline styles are applied directly to an element in your HTML code. They use the style attribute, followed by regular CSS properties
HTML files with an internal stylesheet include a set of rules in their head section. CSS rules are wrapped in style tags.
An HTML page styled by an external CSS stylesheet must reference the .css file in the document head. Once you create the CSS file, it must be uploaded to your server and linked in the HTML file with code.

Q3 Justify-content allows you to do what?

The justify-content property aligns the flexible container's items when the items do not use all available space on the main-axis (horizontally).

Q4 Difference between absolute and relative positioning

Ans Relative position allows positioning according to it's normal position whereas absolute position allows position allows according to it's first parent element.

Q5 What is grid-template-columns used for?

The grid-template-columns property specifies the number (and the widths) of columns in a grid layout. The values are a space separated list, where each value specifies the size of the respective column.

Q6 What is the z-index in CSS?

Ans The z-index property specifies the stack order of an element.
An element with greater stack order is always in front of an element with a lower stack order.
z-index property will not work if the position is static.

Q7 What is the difference between padding and margin?

Margin is the space around an element's border, while padding is the space between an element's border and the element's content.

Q8 What is box sizing?

The CSS box-sizing property allows us to include the padding and border in an element's total width and height.
By default, the width and height of an element is calculated like this:
width + padding + border = actual width of an element
height + padding + border = actual height of an element

Q9 What is animation delay?

Ans: The animation-delay property specifies a delay for the start of an animation.
Negative values are also allowed. If using negative values, the animation will start as if it had already been playing for N seconds.

Q10 Which property you will use to merge cells vertically in grid?

Ans: Grid Column

Q11 Which property you will use to merge cells horizontally in grid?

Ans: Grid Row

Q12 Explain CSS box model

Ans: In CSS, the term "box model" is used when talking about design and layout.
The CSS box model is essentially a box that wraps around every HTML element. It consists of: margins, borders, padding, and the actual content.

Q 13 What is the difference between display none and visibility hidden?

display:none means that the tag in question will not appear on the page at all (although you can still interact with it through the dom).
There will be no space allocated for it between the other tags.
visibility:hidden means that unlike display:none, the tag is not visible, but space is allocated for it on the page.
